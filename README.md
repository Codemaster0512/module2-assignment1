# Module 2: Basic Python Concepts - Assignment 1

## Overview
This repository contains solutions for **Module 2 Assignment 1** with two Python programs demonstrating basic Python concepts.

## Task 1: Mathematical Operations Calculator
**File:** `task1.py`

**Features:**
- Takes two numbers as user input
- Performs addition, subtraction, multiplication, and division
- Includes error handling for division by zero
- Displays formatted results


## Task 2: Personalized Greeting Generator
**File:** `task2.py`

**Features:**
- Takes first name and last name as input
- Concatenates names into full name
- Generates personalized greeting message
